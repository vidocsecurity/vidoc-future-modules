## Getting Started with Vidoc

Welcome to Vidoc! Vidoc is a security automation platform that allows you to automate your Recon and Security Testing. Follow this guide to get started with Vidoc.

## 1. Create a Vidoc Account

To get started with Vidoc, you need to create an account. You can create an account by visiting [https://app.vidocsecurity.com](https://app.vidocsecurity.com).

## 2. Provide username and complete onboarding

Once you have created an account, you will be asked to provide a username, it will be used to create your Vidoc subdomain. Once you have provided a username, you will be asked to complete the onboarding process.

## 3. Start your first Recon

If you want to learn about what is Recon go to [Recon Guide](/features/recon). To start your first Recon, click on the `New` button on the top left corner of the screen and select "New Recon".

## 4. Setup Automatic Scans

If you want to learn about what is Automatic Scans go to [Automatic Scans Guide](/features/automated-scan). To setup your first Automatic Scan, click on the `New` button on the top left corner of the screen and select "New Automatic Scan".
