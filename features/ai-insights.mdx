## What are Target insights?

It is a description generated from information that Vidoc was able to gather about given target.

## What information is used to generate insights?

Vidoc gathers information from multiple sources to generate insights.

**Data used to generate insights:**
- Issues detected
- DNS Records
- Web servers
    - Fingerprinting
    - Technology detection
    - HTTP headers
- Open ports
    - Service detection
    - TLS certificate detection
    - Misconfigurations in TLS certificates
