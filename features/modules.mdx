## What is Module?

It is a scanning feature that allows you to execute [Vidoc modules](/docs/vidoc-modules) on a target.
It is a vulnerability scanning tool that uses Vidoc modules to perform the scan. It is a very powerful tool that can be used to scan for vulnerabilities in a wide range of targets.
